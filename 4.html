Overview

Shroom Houses is a 3D environment series exploring whimsical, organic architecture inspired by fungi and fantasy. Each render experiments with lighting, texture, and surreal scale to create immersive, dreamlike dwellings. This project hopes to evoke a sense of nostalgia using liminal spaces as reference.
